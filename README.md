#  Features
##  Medal Tally

- View total medals won by each country.

- Filter by year and country to analyze specific Olympic Games or nations.

##  Overall Analysis

- Explore top statistics such as number of editions, hosts, sports, events, and athletes.

- Visualize how nations, athletes, and events have evolved over time.

- Heatmaps for event distribution across sports and years.

- Identify the most successful athletes in each sport.

#  Country-Wise Analysis

- Track a country's medal performance over the years.

- Visualize the sports in which a country excels.

- Discover the top athletes from each country.

##  Athlete-Wise Analysis

- View age distribution of medal winners (Gold, Silver, Bronze).

- Compare athlete Height vs. Weight by sport and gender.

- Analyze male vs. female participation trends throughout Olympic history.

##  Technologies Used

- Python 

- Streamlit – Interactive web framework

- Pandas – Data manipulation

- Plotly – Dynamic visualizations

- Matplotlib & Seaborn – Statistical plots

- CSV Datasets – athlete_events.csv, noc_regions.csv
